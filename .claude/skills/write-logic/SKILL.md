# Refine Logic

Improve:

- transitions
- argument coherence
- cause-effect clarity

## Focus

- does each paragraph logically lead to next?
- are assumptions explicit?
- are arguments tight?

## Output

Update:

- DRAFT.md

## Anti-patterns

- rewriting everything blindly
- adding new ideas instead of improving logic
